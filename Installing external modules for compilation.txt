
1) Download IoC framework from here: https://github.com/ybainier/Hypodermic

Go in "Configuration Properties > C/C++ > General > Additional Include Directories" and include there Hypodermic root folder(this folder should contain: bin, Hypodermic, lib, obj, resources, Hypodermic.Test folders).

2) To install Boost download it from here: https://www.boost.org/users/history/version_1_73_0.html

Then extract files in some folder. 
Use this guide to connect Boost to Visual Studion project: https://www.boost.org/doc/libs/1_55_0/more/getting_started/windows.html#build-from-the-visual-studio-ide

3) Use this guide to install SFML: https://ravesli.com/graficheskaya-biblioteka-sfml-vstuplenie-i-ustanovka/

P.S. If you get C4996 in Visual Studio, use this: https://stackoverflow.com/a/25046975
